Welcome to Armed Geniuses

Тут ми обговоримо, як налаштувати вашу лабораторію тестування на проникнення для практичних цілей.

Використання віртуальних машин:


Завантаження Oracle Virtual box

Вам знадобиться програмне забезпечення, яке міститиме всі ваші віртуальні машини та створюватиме пісочничне середовище. Для цього ми будемо використовувати віртуальний ящик oracle. Ви можете завантажити та встановити за посиланням нижче:



Завантаження Kali Linux

Вам знадобиться kali linux або будь-який інший дистрибутив Linux зі спеціальними інструментами встановлено. Для kali linux натисніть посилання нижче. Вам також знадобиться адаптер WiFi, сумісний з kali-linux, для режиму монітора та перехоплення/введення пакетів.

Завантаження Windows VM

Вам також знадобляться машини, щоб діяти як ваша ціль. Ми будемо використовувати Windows-машину діяти як звичайний користувач у реальних ситуаціях.
Ви можете завантажити їх за посиланням нижче.

Завантаження Metasploitable
